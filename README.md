# amssoft-geoip-firewall
Instalador e gerenciador automatizado de firewall GeoIP para servidores Linux. Bloqueie ou permita tráfego por país com whitelist/blacklist, proteção de domínios (APIs, gateways de pagamento) e redes confiáveis (Cloudflare, Fastly). Baseado em geoip-shell + nftables/iptables.
